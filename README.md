이 **Otodom Poland Real Estate Listings Dataset Sample**에는 **6,252개 레코드**가 포함되어 있으며, **Bright Data API**를 사용하여 추출되었습니다.

## Data Points

| Field Name | Description |
|------------|-------------|
| `timestamp` | 데이터셋에서 리스팅이 기록된 날짜 및 시간입니다. |
| `title` | 부동산 리스팅을 설명하는 간단한 제목으로, 종종 주요 특징을 강조합니다. |
| `price` | 등록된 부동산의 매도 희망가이며, 폴란드 즈워티(PLN) 기준입니다. |
| `market` | 1차 시장 또는 2차 시장 등 시장 유형을 나타냅니다. |
| `surface` | 제곱미터 단위의 부동산 총 거주 면적입니다. |
| `location` | 도로명, 도시, 주(보이보드십)를 포함한 부동산의 전체 주소입니다. |
| `remote_support` | 잠재 구매자를 위한 어떤 형태의 원격 지원을 제공하는지 여부를 나타냅니다. |
| `lighting` | 부동산의 자연광 또는 인공조명 특징에 대한 상세 정보입니다. |
| `advertiser_type` | 리스팅이 부동산 중개업체에 의해 게시되었는지 개인 판매자에 의해 게시되었는지 지정합니다. |
| `description` | 부동산 및 그 특징에 대한 상세 설명입니다. |
| `no_of_rooms` | 부동산에 포함된 총 방 개수입니다. |
| `form_of_property` | 아파트, 주택 등 부동산의 유형 또는 분류입니다. |
| `url` | 전체 부동산 리스팅을 확인할 수 있는 웹 주소입니다. |
| `is_for_sale` | 현재 해당 부동산이 판매 중인지 여부를 나타내는 boolean 값입니다. |
| `posting_id` | 데이터셋에서 부동산 리스팅을 식별하기 위한 고유 식별자입니다. |


## Use Cases

### 1. Real Estate Market Analysis
폴란드 부동산 시장의 추세를 분석하여 투자 기회와 시간 경과에 따른 가격 변동을 파악합니다.

### 2. Competitor Analysis for Agencies
부동산 중개업체는 이 데이터를 사용하여 경쟁사 대비 벤치마킹을 수행하고 리스팅 전략을 정교화할 수 있습니다.

### 3. Urban Development Research
연구자는 도시 개발이 지역별 부동산 가치와 주택 공급 가능성에 미치는 영향을 연구할 수 있습니다.
