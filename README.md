이 **Otodom Poland 부동산 매물 Dataset 샘플**에는 **6,252개 레코드**가 포함되어 있으며, **Bright Data API**를 사용해 추출되었습니다.

## 데이터 포인트

| 필드 이름 | 설명 |
|------------|-------------|
| `timestamp` | dataset에 매물이 기록된 날짜와 시간입니다. |
| `title` | 부동산 매물을 설명하는 간단한 제목으로, 주요 특징을 강조하는 경우가 많습니다. |
| `price` | 등록된 부동산의 희망 가격으로, 폴란드 즈워티(PLN) 기준입니다. |
| `market` | 1차 시장 또는 2차 시장과 같은 시장 유형을 나타냅니다. |
| `surface` | 제곱미터 단위의 부동산 총 주거 면적입니다. |
| `location` | 거리, 도시, 주를 포함한 부동산의 전체 주소입니다. |
| `remote_support` | 해당 매물이 잠재 구매자를 위한 어떤 형태의 원격 지원을 제공하는지 나타냅니다. |
| `lighting` | 부동산의 자연광 또는 인공조명 관련 특징에 대한 세부 정보입니다. |
| `advertiser_type` | 해당 매물이 중개업체에 의해 게시되었는지 또는 개인 판매자인지를 명시합니다. |
| `description` | 부동산과 그 특징에 대한 상세 설명입니다. |
| `no_of_rooms` | 해당 부동산에서 이용 가능한 총 방 개수입니다. |
| `form_of_property` | 아파트 또는 주택과 같은 부동산 유형 또는 분류입니다. |
| `url` | 전체 부동산 매물을 확인할 수 있는 웹 주소입니다. |
| `is_for_sale` | 해당 부동산이 현재 판매 중인지 여부를 나타내는 boolean 값입니다. |
| `posting_id` | dataset 내 부동산 매물의 고유 식별자입니다. |


## 활용 사례

### 1. 부동산 시장 분석
폴란드 부동산 시장의 트렌드를 분석하여 투자 기회와 시간에 따른 가격 변화를 파악합니다.

### 2. 중개업체를 위한 경쟁사 분석
부동산 중개업체는 이 데이터를 사용해 경쟁사와 비교 벤치마킹하고 매물 등록 전략을 개선할 수 있습니다.

### 3. 도시 개발 연구
연구자들은 도시 개발이 지역별 부동산 가치와 주택 공급 가능성에 미치는 영향을 연구할 수 있습니다.
